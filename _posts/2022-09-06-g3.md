## 1. (교육용) Web IDE 가 갖추어야 할 요소

- 웹 브라우저로 IDE 에 액세스 할 수 있어야 함.
- 한 화면에서 아래 요소들이 적절히 노출되어야 함.
  - (1) 자습서 (학습 내용, 실습 내용)
  - (2) 에디터
  - (3) 실행 화면
  - (4) 네비게이터
  - (5) 실습 도구(힌트, 초기화, 저장, 다운로드, 제출 등)  

- Web IDE → Cloud IDE 로 옮겨지는 트렌드에 따라, 코드가 공유될 수 있어야 함.
